\# VOIDSTAMP v3.3 | Digital Footprint Eraser



!\[License](https://shields.io)

!\[Python](https://shields.io)



\*\*VOIDSTAMP\*\* is a high-performance terminal-based utility designed for deep cleansing of digital traces within Telegram. It leverages the Telethon library to automate the destruction of outgoing metadata, media, and communication logs.



\## ⚡ Features

\- \*\*Temporal Erasure\*\*: Wipe data starting from a specific point in time.

\- \*\*Granular Targets\*\*: Delete Video Rounds (circles), Voice Messages, or Plain Text separately.

\- \*\*Total Wipeout\*\*: A nuclear option to clear all outgoing data for everyone (`revoke=True`).

\- \*\*Node Cleanup\*\*: Automatically leave or delete inactive chats (30+ days of silence).

\- \*\*Secure Vault\*\*: Credentials are encrypted and stored locally.
